Modules

You can search directly to your supplier for ingredients by intergrating its API here.

For example, given your supplier provides a valid API to return its products, you can write simple modules to import the data directly to your local PV database.

Structure

  • supplier1.json
  • supplier1.php

JSON file contains the supplier data needed to register with PV

eg:

{"id":"supplier_id","name":"My custom Supplier","slug":"mycustomsupplier","version":"1.0","fileName":"supplier1.php","icon":"fas fa-database"}

PHP file should contain the code that query's the supplier's API and return the results in JSON format

eg:
