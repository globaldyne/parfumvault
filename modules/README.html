Modules

You can search directly to your supplier for ingredients by integrating its API here.

For example, given your supplier provides a valid API to return its products, you can write simple modules to import the data directly to your local PV database.

Structure

  • supplier1.json
  • supplier1.php

The JSON file contains the supplier data needed to register with PV, e.g.:

{
  "id": "supplier_id",
  "name": "My custom Supplier",
  "slug": "mycustomsupplier",
  "version": "1.0",
  "fileName": "supplier1.php",
  "icon": "fas fa-database"
}

The PHP file should contain the code that queries the supplier's API and returns the results in JSON format, e.g.:

{
  "draw": 1,
  "recordsTotal": 1,
  "recordsFiltered": 1,
  "data": [
    [
      "ambermax (Givaudan) - 50% TPM",
      
      ...

      "Perfumers Apprentice",
      
      ...
    ]
  ]
}
